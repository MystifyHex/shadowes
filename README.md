# What is this?

 A shade for your webpage

# Installation

`npm install shadow`

Then...

```
import { shadow } from 'shadow';

shadow({
    shadow_type: 'soft',
    padding: 'false'
});
```

## Options

Shadow supports 2 options, both of which are optional:

* *shadow_type* - _hard | soft_ (Defaults to soft)
* *padding* - _boolean_ (Defaults to false)
